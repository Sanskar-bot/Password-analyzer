CUPP — Common User Passwords Profiler
The personalized attack simulator is inspired by CUPP (by Mebus / j0rgan), a security research tool that generates targeted password dictionaries from personal information. The JavaScript implementation in this extension is a clean-room rewrite: no Python code, no file I/O, no network calls.
