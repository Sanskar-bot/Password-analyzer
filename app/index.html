Process

How PassGuard works

Every analysis runs a multi-layer pipeline locally in your browser.

1
Input
Enter a password. Optionally add your username, or run the personalized attack with profile data. Nothing leaves your device.
2
Analysis Pipeline
Six modules run in parallel: strength, patterns, wordlist, username similarity, brute-force time, and personalized dictionary scoring.
3
Composite Score
A weighted composite score (0–100) is calculated, combining all signals. Each component is shown transparently in the breakdown.
4
Recommendations
Specific, actionable suggestions are generated based on detected weaknesses — not generic advice.
